# Watermark Generator (Tkinter + Pillow)

A simple, beginner-friendly desktop tool to add a **semi-transparent text watermark** to any image.

Perfect for quickly protecting your photos before sharing them online.

## Features

- Browse and load any image (JPG, PNG, JPEG)
- Enter custom watermark text
- Automatically places the watermark in the **center** of the image
- Semi-transparent white text (opacity ~50%)
- Preview the watermarked image instantly
- Save the final watermarked image to your computer
- Clean, minimal Tkinter interface with a preview canvas
